# YouTube Data Analysis
#### Discover which YouTube categories are growing the fastest and what drives their success. 

Is it the having the highest number of subscribers? Is it content optimization ? Is it the frequent uploading which works out? Lets dive in..... 

## Overview
This project focuses on analyzing YouTube channel performance data to extract meaningful insights related to views, engagement, upload frequency, and content efficiency. The objective is to identify patterns that influence channel growth and monetization potential using data-driven techniques.

## Analysis Outputs
--1.png – Contains all the visualizations generated during the Exploratory Data Analysis (EDA)

--2.png – Contains the business insights derived from the data analysis. Includes Key takeaways , decision rules and strategic recommendations for content optimization and monetization.
